# DXF Sketcher 1.5.1

DXF Sketcher 1.5.1 is a usability and workflow release focused on everyday DXF jobs.

## Highlights

- New project system with `.dxsp` files for saving and reopening full multi-file jobs
- Sidebar upgrades for multi-folder browsing, visibility toggles, color markers, and direct rename actions
- Beginner-friendly tool hints and cleaner toolbar popovers
- Better Light and Light-Blue theme readability
- `Convert to geometry` workflow for making cluster content editable in place
- Linux release packaging in `.deb`, `.rpm`, and `.AppImage` formats

## Included Artifacts

- Debian package: `dxfsketcher_1.5.1_*.deb`
- RPM package: `dxfsketcher-1.5.1-1.*.rpm`
- AppImage: `dxfsketcher-1.5.1-*.AppImage`

## Notes

- Linux is the main release target for 1.5.1
- Windows and macOS builds will follow a bit later while a few platform-specific issues are still being resolved
- `DXF` remains the main editable document workflow
- `SVG` export is available through `Save As`
