# DXF Sketcher

DXF Sketcher — это 2D-редактор DXF для быстрой и понятной работы с эскизами.

Проект сделан как форк [dune3d](https://github.com/dune3d/dune3d), но интерфейс и сценарии работы переработаны под более простой ежедневный DXF-поток.

Репозиторий: <https://github.com/EriArk/-DXF-Sketcher>

## Что важно пользователю

- Открытие одного или нескольких DXF-файлов
- Открытие целой папки с DXF
- Быстрое редактирование активного эскиза
- `Save` в связанный файл, `Save As` в `DXF` или `SVG`
- Удобные инструменты для повседневных задач: контур, прямоугольник, круг/дуга, полигон, текст, импорт изображения

## Два подхода работы

В DXF Sketcher сохранен **параметрический подход**:
- ограничения, размеры и точная геометрия на месте.

И добавлен более **прямой (частично объектный) подход**, ближе к графическим редакторам:
- быстрее выбирать, двигать и править объекты прямо на холсте;
- меньше лишних действий в типовых задачах.

Главное: вы сами выбираете, каким подходом пользоваться в каждый момент — строгой параметрикой, быстрым прямым редактированием или их комбинацией.

## Быстрый сценарий

1. Нажмите `Open file` или `Open folder`.
2. Выберите нужный sketch.
3. Отредактируйте геометрию.
4. Сохраните через `Save` или `Save As`.

## Текущие ограничения

- Основной формат открытия для редактирования: `DXF`.
- SVG как отдельный документ через `Open file` пока не поддерживается.
- SVG можно импортировать в эскиз через `Import Picture`.
