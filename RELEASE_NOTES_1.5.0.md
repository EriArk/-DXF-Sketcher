# DXF Sketcher 1.5.0 Prerelease

This prerelease is focused on practical DXF editing and fabrication workflows.

## Highlights

- Self-contained **Boxes** integration with bundled `boxes.py`
- Full **Boxes** catalog with categories, favorites, gallery view, sample-photo previews, and native template settings
- New **Edge Features** workflow for joints, hinges, grooves, lids, flex cuts, handles, and utility edge operations
- Built-in **Import Raster** and **Trace Image** workflow for raster-to-vector conversion
- New **Gears** generator for involute gear geometry
- Packaging polish for shipping self-contained release builds

## Included Artifact

- Debian package: `dxfsketcher_1.5.0_*.deb`

## Notes

- This is a prerelease build intended for wider testing.
- `DXF` remains the main editable document workflow.
- `SVG` export is supported through `Save As`.
- Some advanced Edge Features families will continue to improve after this prerelease.
